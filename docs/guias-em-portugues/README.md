# Guias em Portugues

Esta pasta transforma o material do repositorio em leitura rapida, sem copiar linha por linha os prompts originais.

## Conteudo

- `resumo-geral.md`
  Visao consolidada do que cada prompt ou componente faz.

- `fichas-de-arquivos.md`
  Explicacao em portugues, arquivo por arquivo, com foco em entendimento rapido.

- `mapa-por-tema.md`
  Separacao por seguranca defensiva, ofensivo-correlata, permissao, trust e execucao.

- `revisao-de-cobertura.md`
  Auditoria do que entrou, do que faltava e do que continua ausente da amostra original.

Use estes arquivos como entrada principal. Os artefatos brutos ficam nas outras pastas.
