# Versoes Em Portugues

Esta pasta existe para deixar o repositorio facil de entender sem precisar ler primeiro os prompts e arquivos tecnicos originais.

## O que voce encontra aqui

- `prompts-publicos-explicados.md`
  Explica em portugues os prompts publicos mais importantes para seguranca ofensiva e defensiva.

- `src-zip-explicado.md`
  Traduz para linguagem clara o que os arquivos centrais do `src.zip` fazem.

- `colecao-seguranca-explicada.md`
  Resume toda a colecao ampliada por tema: review, shell, permissoes, trust remoto e superficie de prompt.

## Como usar

1. Comece em `prompts-publicos-explicados.md`.
2. Passe para `src-zip-explicado.md`.
3. Feche em `colecao-seguranca-explicada.md`.

Os arquivos originais continuam no repositorio para referencia tecnica. Esta pasta e a camada de entendimento em PT-BR.
